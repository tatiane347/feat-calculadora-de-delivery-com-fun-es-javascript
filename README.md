📝 Descrição do Projeto

Este projeto simula uma Calculadora de Pedidos de Delivery inteligente em tempo real para a Padaria Doce Sabor. O foco central da aplicação é aplicar e demonstrar o domínio de conceitos de Funções em JavaScript, dividindo lógicas de cálculo, condicionais, retornos de fluxo e o uso moderno de Arrow Functions.

O sistema calcula em tempo real o preço dos itens, taxas de entrega com base na quilometragem e o valor de impostos, além de analisar se o cliente se qualifica para entrega gratuita.

🛠️ Funções Implementadas1. Funções Básicas para Cálculos (Atividade 1)

exibirBoasVindas(nome): Retorna uma saudação amigável personalizada com base no nome do cliente digitado no formulário.

calcularSubtotal(preco, quantidade) (Arrow Function): Multiplica o valor unitário pela quantidade de itens.

calcularDesconto(valor, percentual) (Arrow Function): Aplica um percentual de redução sobre o valor fornecido.

calcularTaxaEntrega(distancia): Calcula o valor da taxa baseado na distância (R$ 2,50 por km), estabelecendo uma taxa mínima segura de R$ 5,00.

2. Funções com Retorno (Atividade 2)

calcularImposto(subtotal) (Arrow Function): Retorna o valor fixo de imposto municipal correspondente a 8% sobre o subtotal.

verificarFreteGratis(subtotal): Compara se o subtotal supera R$ 50,00, retornando um booleano (true/false) para controle de isenção de taxa de entrega.

calcularTotalFinal(subtotal, desconto, imposto, taxaEntrega): Retorna a soma líquida total após as deduções e taxas devidas.
